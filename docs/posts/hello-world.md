# 欢迎来到我的博客 🎉

这是我的第一篇博客文章！感谢你的访问。

## 关于这个博客

这个博客使用 **VitePress** 构建，具有以下特点：

- ⚡️ **快速构建** - 基于 Vite，开发体验极佳
- 📝 **Markdown 优先** - 用 Markdown 写作，简单高效
- 🎨 **美观主题** - VitePress 默认主题，支持深色模式
- 🔍 **SEO 友好** - 静态站点，搜索引擎友好
- 📱 **响应式设计** - 支持手机、平板、桌面端

## 如何发布新文章

1. 在 `docs/posts/` 目录下创建新的 `.md` 文件
2. 文件名会成为文章的 URL 路径
3. 文章会自动出现在侧边栏中

```python

def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```
